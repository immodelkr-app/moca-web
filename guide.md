# 프로젝트: 광고모델 에이전시 투어 매니저 (Agency Tour Manager)
# 목표: 광고 모델들이 에이전시 위치를 확인하고, 방문 전 체크리스트를 수행하며, 투어 기록을 남기는 앱 개발


앱은 4개의 탭으로 구성됩니다.
1. Agency Map (지도): 에이전시 위치 확인 및 정보 제공
2. Tour Checklist (체크리스트): 방문 전 의상 및 상태 점검
3. Tour Diary (다이어리): 방문 기록 및 메모
4. I'M TV & Shop (콘텐츠): 유튜브 영상 교육 및 추천 아이템 쇼핑




- 기능: Google Maps(또는 사용 가능한 지도 API)를 전체 화면에 표시.
- 데이터 연동: 함께 업로드된 `agencies.csv` 파일을 로드하여 지도에 마커(Marker) 표시.
- 인터랙션:
  - 마커 클릭 시 하단에 'Bottom Sheet'가 올라옴.
  - Bottom Sheet 내용: 에이전시 이름(Name), 주소(Address), 카테고리(Category).
  - '길찾기' 버튼을 누르면 외부 지도 앱으로 연결 (옵션).


- 목적: 모델이 에이전시 방문 전 준비 상태를 확인.
- 섹션 1 (전략 설정):
  - 나의 이미지 키워드 입력 (Text Input)
  - 목표 카테고리 선택 (Dropdown)
- 섹션 2 (오늘의 의상 체크 - 필수):
  - [이미지 업로드 버튼]: '상반신', '전신' 사진을 찍어서 등록.
  - [체크박스 리스트]:
    1. 에이전시 성향에 맞는 의상인가?
    2. 헤어/메이크업은 깔끔한가?
    3. 프로필 PPT 지참 완료?
  - '저장' 버튼을 누르면 로컬 저장소에 날짜와 함께 저장.


- UI: 달력(Calendar) 뷰가 기본.
- 기능:
  - 날짜 선택 시 해당 날짜의 방문 기록 표시.
  - [기록 추가] 버튼: 방문한 에이전시 선택, 메모(담당자 반응 등) 입력.
  - 데이터는 로컬 DB(SQLite 등)에 저장하여 앱을 껐다 켜도 유지되게 함.

- 유튜브 섹션: '아임모델 김대표TV' 채널의 영상을 보여줌 (WebView 또는 Youtube API).
- 쇼핑 섹션: 모델에게 필요한 패션/뷰티 아이템 추천 리스트 (이미지 + 구매 링크 버튼).


- 프레임워크: Flutter (또는 사용 중인 노코드 툴 환경에 맞춤)
- 데이터 소스:
  - 에이전시 정보: `agencies.csv` (포함된 파일)
  - 사용자 데이터: 로컬 디바이스 저장소 (SharedPreferences 또는 SQLite)


- 전체적으로 깔끔하고 전문적인 느낌 (Black & White 톤).
- 버튼은 직관적이고 크게 배치 (현장에서 급하게 쓸 수 있도록).
