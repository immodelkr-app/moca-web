# 🔗 현재 접속 주소 네이버에 추가하기

네, 좋은 방법입니다!
사용자님이 자꾸 접속하게 되는 **그 주소(Vercel 임시 주소)**를 네이버에 등록해버리면 편합니다.

## 따라해보세요
1. 지금 **지도가 안 나오는 페이지**의 인터넷 주소창을 클릭합니다.
2. 주소 전체를 **복사**합니다.
   - (예시: `https://im-model-7tpsd7we3-immodelkr-7983s-projects.vercel.app`)
3. 네이버 클라우드 콘솔 > Application > **Web 서비스 URL** 설정으로 갑니다.
4. 복사한 주소를 **추가**하고 저장합니다.
   - (팁: 주소 끝에 `/`를 붙인 것도 하나 더 추가하면 완벽합니다)

이렇게 등록하시면, 5~10분 뒤부터는 그 주소로 접속해도 지도가 잘 나옵니다!
